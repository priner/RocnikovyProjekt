Ako používať?

Program je treba spúšťať z priečinku v ktorom sa nachádza.
Prvý argument je cesta ku grafu.
Druhý je cyklus (čísla vrcholov oddelene čiarkamy, bez medzier).
Program uloží vysledky do súboru, čísla vrcholov v cykle pridá ako suffix
k názvu súboru.
Ak nešpecifikujete žiaden cyklus, program len vypíše všetky 5-cykly v grafe.

Príklad:

python3 5cycle.py ../Grafy/5FLOWE3.34 1,2,5,8,4
